Help
New Project Wizard
Create a project in 4 easy steps:
  • Step 1: Enter the project name, dates, and description.
  • Step 2: Choose or create the client.
  • Step 3: Select which engagement types are part of this project.
  • Step 4: Review everything and create the project.
Each selected engagement type creates a planned engagement linked to this project. You can configure each engagement individually after creation.
