Projects
Projects group multiple engagements under one umbrella. A client may award a project that includes external pen testing, web app pen testing, WiFi audit, and more.
  • Name: The project title — click to view details.
  • Client: The client for this project.
  • Status: Active, On Hold, or Completed.
  • Engagements: Number of engagements linked to this project.
  • Dates: Start and end dates for the project timeline.
