Edit Engagement
Update an existing engagement's details and configuration.
  • Name: A descriptive name for this engagement (e.g. "Q1 External Pentest").
  • Client: The owning client (read-only after creation).
  • Status: Current phase — Planned, Active, On Hold, or Completed.
  • Start Date: When the engagement begins.
  • End Date: Optional expected completion date.
  • Description: Scope and objectives for this engagement.
  • Notes: Internal notes — constraints, delivery details, etc.
