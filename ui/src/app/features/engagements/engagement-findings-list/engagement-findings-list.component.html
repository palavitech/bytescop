Findings
Findings represent issues discovered during testing. Keep severity and status consistent for reporting.
  • Timeline: The progress bar shows the engagement's time remaining based on start and end dates.
  • Draft: Findings saved as draft are marked with a "Draft" badge. They can be published later from the edit screen.
  • Filters: Narrow by severity and status using the Filter pane.
  • Assets: Each finding is linked to an in-scope asset for traceability.
  • Updated: Track recency for remediation follow-ups.
  • New Finding: Create a finding linked to an in-scope asset. The SoW must be approved before findings can be created.
