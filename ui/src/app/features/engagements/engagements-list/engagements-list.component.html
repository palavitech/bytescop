Engagements
Engagements represent security assessment projects. Track delivery timelines, scope, and status. Click a row name to open the detail view.
  • Name: The engagement title — click to view details.
  • Client: The client for this engagement.
  • Status: Planned, Active, On Hold, or Completed.
  • Summary: Severity pills showing the count of Critical, High, Medium, Low, and Info findings.
  • Dates: Start and end dates for the engagement timeline.
  • Findings: Click the Findings button to jump directly to an engagement's findings list.
  • Filters: Narrow results by client or status using the Filter panel.
  • Export: Use the Actions menu to export the list as CSV.
