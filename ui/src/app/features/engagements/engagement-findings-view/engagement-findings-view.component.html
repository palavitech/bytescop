View finding details including severity, status, classification, description, and remediation recommendations.
  • Draft: Findings saved as draft show a "Draft" badge. Edit to publish when ready.
  • Severity: Impact rating from Info to Critical.
  • Status: Current triage state — Open, Triage, Accepted, Fixed, or False Positive.
  • Asset: The in-scope asset this finding is linked to.
  • Classification: Shows Assessment Area, OWASP Top 10 category, and CWE identifier when set.

  • Recommendation: Suggested remediation steps (if provided).
  • Edit: Update the finding details and markdown content.
  • Delete: Permanently remove the finding from this engagement.
