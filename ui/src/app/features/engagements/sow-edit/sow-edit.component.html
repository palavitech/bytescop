Edit Statement of Work
Update the statement of work for this engagement.
  • Title: A descriptive title for the SoW document.
  • Status: Use the Approve / Revert button in the status bar to toggle between Draft and Approved.
  • Scope lock: When approved, scope assets are locked — you cannot add or remove assets. Revert to Draft to unlock scope changes.
  • Findings gate: The SoW must be approved before findings can be created for this engagement.
  • Scope Assets: Add or remove assets that are in scope. Only assets belonging to the engagement's client are available in the dropdown.
