Clients
Clients represent your customers. Engagements and assets are grouped under clients.
  • Create: Add a new client with name, website, and notes.
  • Status: Active clients can have new engagements. Inactive ones are archived.
  • Engagements: Click the Engagements button on a row to view that client's active engagements.
  • Export: Use the Actions menu to export the list as CSV.
