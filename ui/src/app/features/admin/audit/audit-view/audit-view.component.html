Audit Entry
Full details of a single audit log entry.
  • Details: Metadata about the action — timestamp, action type, resource, user, IP address, request path, request ID, and user agent.
  • Changes: For update actions, shows a diff table with which fields changed, including old and new values side by side.
  • Snapshots: Collapsible before/after JSON snapshots of the resource state. Changed fields are highlighted (red for removed, green for added).
