Audit Log
The audit log tracks all actions performed in this tenant, including resource changes, authentication events, and admin operations.
  • Visualize: Click Visualize to view interactive charts showing activity trends, action distribution, resource breakdowns, and top actors. Click any bar to filter. Charts react to active filters.
  • Filters: Use the filter panel to narrow results by user, action type, resource type, date range, resource ID, or IP address.
  • Detail View: Click a timestamp to see full details including before/after snapshots and field-level diffs.
  • Pagination: Navigate between pages using the controls below the table. Change page size in the header bar.
  • Read-only: Audit logs cannot be modified or deleted.
