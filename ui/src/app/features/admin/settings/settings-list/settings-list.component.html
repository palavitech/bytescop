Help
Settings
Settings control tenant-wide configuration for your organization. Changes take effect immediately.
  • Text settings: Type a value and click the save button to apply.
  • Boolean toggles: Click the toggle to enable or disable. Changes save automatically.
  • Choice dropdowns: Select an option and click save to apply.
  • Logo: Upload a JPEG, PNG, WebP, or GIF up to 1 MB. Shown in reports and branding.
  • Reset: Click the reset button to revert a setting to its default value.
  • License: Enter a license key to unlock enterprise features. Keys are validated offline — no internet required. Purchase at bytescop.com.
  • Danger Zone: Permanently close and delete this tenant. This is irreversible and requires MFA verification plus an email confirmation code. Back up your database first using ./backup.sh.
  • Audit: All changes are recorded in the Audit Log.
