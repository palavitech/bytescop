Groups
Groups bundle permissions together and assign them to users. Default groups come pre-configured with standard permission sets.
  • Default groups: Pre-configured groups (e.g. Administrators, Analysts, Collaborators) are created automatically. They cannot be deleted or have their permissions changed, but you can manage their members.
  • Custom groups: Create your own groups with specific permission sets tailored to your workflow.
  • Members: Add or remove users from groups on the group edit page.
  • Permissions: Each group can have any combination of the available permissions.
  • Export: Use the Actions menu to export the group list as CSV.
