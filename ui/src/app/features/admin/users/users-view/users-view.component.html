View user details, role, and group memberships.
  • Edit: Update user details and group assignments.
  • Delete: Remove the user from this tenant (owners cannot be deleted).
  • Groups: Shows which permission groups the user belongs to.
  • Promote: Owners can promote a member to owner (requires your MFA code).
  • Demote: Owners can demote another owner to member (cannot demote yourself).
  • Reset MFA: Owners can reset another user's MFA, requiring them to re-enroll on next login.
  • Status: Shows Active, Locked, or Invited (pending invite acceptance).
  • Engagement Assignments: Shows which engagements this user is a stakeholder on. Add or remove assignments directly.
