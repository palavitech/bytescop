Create User
Add a new user to this tenant.
  • Email: If the email matches an existing account, the user is added to this tenant. Otherwise a new account is created.
  • Name: First and last name for the new user.
  • Password: Set a temporary password for the user. Share it with them securely.
  • Role: All new users are assigned the Member role. Only the tenant signup owner gets the Owner role.
  • Groups: Assign permission groups to control what the user can access. You can change these later.
