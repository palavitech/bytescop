Users
Manage the users who have access to this tenant. Assign roles and groups to control what each user can do.
  • Roles: Owner and Member. Owners can promote members or demote other owners from the user detail page.
  • Groups: Assign users to permission groups to grant fine-grained access.
  • Invite: New users receive an email invitation to set their password and join the tenant.
  • Lock/Unlock: Temporarily disable a user's access without removing them. Owners cannot be locked.
  • MFA: Shows whether each user has multi-factor authentication enabled.
  • Export: Use the Actions menu to export the user list as CSV.
