Assets
Assets represent the systems, applications, and infrastructure being assessed. They can be grouped under clients.
  • Type: Host, WebApp, API, Cloud, Network Device, Mobile App, or Other.
  • Environment: Prod, Staging, Dev, or Lab.
  • Criticality: Low, Medium, or High — indicates business impact.
  • Target: IP address, hostname, URL, or other identifier.
  • Client: The client that owns this asset. Click the client name to navigate to it.
  • Filter: When accessed from a client page, the list is pre-filtered to that client's assets. Use "Clear Filter" to show all.
  • Export: Use the Actions menu to export the list as CSV.
