Create Asset
Create a new asset to track in your security assessments.
  • Name: A descriptive name for the asset.
  • Client: Assign the asset to a client. Pre-selected if created from a client page.
  • Type: The kind of asset — Host, WebApp, API, Cloud, Network Device, Mobile App, or Other.
  • Environment: Prod, Staging, Dev, or Lab.
  • Criticality: Low, Medium, or High — indicates business impact.
  • Target: The IP, hostname, or URL to assess.
  • Notes: Any additional information about this asset.
